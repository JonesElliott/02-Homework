Super Asteroids is a demo of an Asteroids style game that I created using Game Maker Studio 2 utilizing their programming language, Game Maker Language. This project took me about a month in my spare time to complete from scratch. Super Asteroids represents a milestone in my programming hobby and has given me a lot of confidence in what I have learned and makes me excited for future endeavors.
